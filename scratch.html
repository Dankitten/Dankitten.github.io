Name Project

Description:

In this project, I used Scratch to design and create a program that shows my name. The requirements were to add a sprite for each letter in my name and to give them at least 3 unique behaviors between the letters. The program needed to run when the green flag was clicked, and for extra credit, the program would reinitialize when the green flag was clicked.

Concepts Learned

  • Using Event Blocks
  • Conditional Statements
  • Loops
  • Sprites
  • Movement

Computer Buying Project

Description:

Make a presentation using Scratch about the parts of a computer, compare and contrast the pros and cons of various computers on the market, and recommend a computer from a selection of computers based on their properties. I used a counter variable that recorded the slide number. Sprites would show or hide based on what the slide number was, creating an effect similar to a slideshow.

Concepts Learned

  • Using Variables
  • Using the Background
  • Showing and Hiding Sprites
  • Parts of a computer and their function

Final Project

Description:

Using Scratch, make an interactive game or movie. I chose to make a game. Games should have a score, utilize the timer, have a help section, and tell the user when they win. My game was based off of beatMania, with blocks falling from the top of the screen to be hit when they touched the corresponding key. A score variable was used to record the score, and different amounts of points were given based on how accurately the player hit the key.

Concepts Learned

  • Lots of Sprites working in tandem
  • Copies
  • Background
  • User Input
  • Conditionals
  • Broadcasts
  • Timers
